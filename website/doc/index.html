Obtain the Android SDK

Download the Android SDK from http://developer.android.com/sdk/

Configuring the project from the command line

  1. Run the command android list targets and find the number that belong the Android target you are developing your app for. The minimum Android version supported by Dyad is Android 2.2 (API 8).

  2. Run the command android update lib-project --target {target-number} --path {dir}/dyad/client where {target number} is the number of the previous step and {dir} is the path to the directory where you placed the Dyad source code.

  3. Copy android-support-v4.jar from the directory android-sdk/extras/android/support/v4/ to dyad/client/libs/.

  4. If you want to use Bump for bonding, sign up as a developer.

    5. Importing the project into Eclipse

    Import the client part of dyad as an Android project into Eclipse. Go to the Project Properties section to change the Android Project into an Android Library.

    Android Support Libraries

    Dyad Client tries to be backward compatible with Android devices back to API level 8. Blablabla

    To properly reference the Android Support libraries in Eclipse: > Project Properties > libraries > Add External Jars Select {path-to-sdk}/extras/android/support/v4/android-support-v4.jar and add the support library jar-file to your buildpath.
